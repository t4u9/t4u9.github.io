基本

HTML(HyperText Markup Language)超文本编辑语言，由元素组成。 元素 开始标签 结束标签 内容 属性 可以拥有 布尔属性 只能拥有和属性名一样的属性值 元素可嵌套 块级元素 在页面中与块的形式展现。通常用来调整结构。 内联元素 不会换行，用来修饰。 空元素 只有一个标签。通常用来插入/嵌入一些东西。 HTML解释器会将连续出现的空白字符减少为一个单独的空格符

特殊字符

< &lt; < < > > " " ' ' & &

head

不会在浏览器中展示 tille 标题 整个文件的标题 meta 元数据 （空元素） 特性 <meta charset="" > 字符编码 name和description 定义作者和页面描述(可在搜索引擎相关页面看到，SEO) <meta name="" content="" > 其他 property 自定义图标 将其保存在与网站的索引页面相同的目录中，以.ico格式保存（大多数浏览器将支持更通用的格式，如.gif或.png） 将以下行添加到HTML 中以引用它： <link rel="shortcut icon" href="favicon.ico" type="image/x-icon"> 使用CSS和JavaScript <link> 空元素 css <script> 不是空元素 最好添加在页面底部 主语言 < html lang="zh-CN"> 部分设置 <span lang="jp"> 搜索引擎更好利用。屏幕阅读器更好识别。
